Input Function

User defined input function to read data from input stream.

UINT in_func (
  JDEC* jdec,    /* Decoder object */
  BYTE* buff,    /* Pointer to buffer to store the read data */
  UINT ndata     /* Number of bytes to read */
);

Parameters

jdec
Specifies the decompression object of the decompression session.
buff
Specifies the pointer to the buffer to store the read data. When a NULL is specified, the read data is discarded.
ndata
Specifies number of bytes to read/skip from the input stream.

Return Values

Returns number of bytes read/skipped. When a 0 is returned, the jd_prepare and jd_decomp function fails with JDR_INP.

Description

This function is the data input interface of the TJpgDec module. The corresponding decompression session can be identified by the pointer to the device identifier jdec->device passed to the 5th argument of jd_prepare function.
